# SE3040 - Application Frameworks: Lab 08

This repository contains the completed exercises for **Lab 08 (React JS - Lab 01)** for the SE3040 Application Frameworks module at SLIIT. 

## Project Overview

This project is a React application initialized using **ViteJS**. It demonstrates the creation and composition of several reusable UI components to build a modern, responsive web application page.

### Implemented Features & Components

The following exercises and components have been successfully implemented:

- **Setup**: Configured ViteJS to run strictly on **Port 3000** and removed default boilerplate code.
- **Exercise 01 (`Greeting`)**: A simple greeting component rendered in the main application.
- **Exercise 02 (`NavBar`)**: A navigation bar component with a brand logo ("TechNova Solutions") and meaningful navigation links.
- **Exercise 03**:
  - **`Footer`**: Displays copyright information and social media links.
  - **`Card`**: A reusable card component to display images, titles, and descriptions for services.
  - **`Button`**: A highly customizable button supporting various styles (`primary`, `secondary`, `outline`, `success`, `danger`) and sizes.
  - **`Banner`**: A promotional hero banner with a call-to-action and background image support with text-overlay gradients.
  - **`Testimonial`**: A component displaying customer quotes, names, and profile photos with smart fallback styling.

## Running the Project Locally

To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/layathmaperera/sliit-se3040-lab08.git
   ```
2. Navigate into the directory:
   ```bash
   cd sliit-se3040-lab08
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:3000`.
